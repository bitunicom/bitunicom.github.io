Introduction

Steps

Select: select optimal child based on upper confidence bound(UCB) recursively until Leaf node L is reached
Expand: if L is not terminal node, create child nodes, select C based on UCB
Simulate/Evaluation: run playout from C
Back Propagation: update path info based on playout result from C up to root node, including visit count.
